# Karyakarsa REST API



## Introduction

Hi, welcome to Karyakarsa REST API, the scraper of karyakarsa creator page

## Usage

 ```
 api.karyakarsa.willo.is-a.dev/mainapi/karyakarsa?name=nicdev
 ```
