# Karyakarsa REST API



## Introduction

Hi, welcome to Karyakarsa REST API, the scraper of karyakarsa creator page

## Usage

 ```
 willo.is-a.dev/karyakarsa-rest-api/mainapi/karyakarsa?name=nicdev
 ```
